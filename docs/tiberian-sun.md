# Tiberian Sun
Below are a list of fields available in the Tiberian Sun 2.03 Game Resolution Packet

### Client Information
##### `VERS` Version
Game client version
* type: `string`
* example: `v2.03`

##### `VIDM` Video Memory
Client's video memory
* type: `int`
* example: `16777216`

##### `MEMO` Memory
Client's available RAM
* type: `int`
* example: `1073741823`

##### `PROC` Processor
Client's processor type
* type: `int`
* example: `6`

##### `AFPS` Average FPS
Client's Average Frames per Second
* type: `int`
* example: `56`

##### `PNGR` Pings Received
Client's Pings Received
* type: `int`
* example: `1`

##### `PNGS` Pings Sent
Client's Pings Sent
* type: `int`
* example: `1`

##### `FINI` Finish
Whether the client saw the finish of the match
* type: `bool`
* example: `1`
 
##### `OOSY` Out of Sync
Whether the match was considered Out of Sync
* type: `bool`
* example: `1`

##### `GSKU` Game SKU
Game SKU
* type: `int`
* example: `4708`
